# Dev Speed Meter

開発作業のスピードをリアルタイムで可視化するCUIツール。
キーボード入力を監視し、スプリントメーターのようにスピードを表示します。

## 概要
```
[██████████████░░░░░░░░░░░░░░░░]  92.3 km/h | RPM:  1234 | 4th  | Power: 23% | Keys: 1234 | Idle:  0.5s | [ZONE]
```

キープレス数（KPS）を開発スピード（km/h）に変換し、
リアルタイムでメーターを表示します。

## インストール
```bash
pip install pynput
```

## 使い方
```bash
python dev_speed_meter.py
```

終了は `Ctrl+C`

## 表示項目

| 項目 | 説明 |
|------|------|
| `[bar]` | スピードを0-180km/hで視覚化 |
| `km/h` | 現在の開発スピード（平滑化） |
| `RPM` | キープレス数 × 30 |
| `Gear` | 自動シフト（スピード連動） |
| `Power` | パワー出力（パーセント） |
| `Keys` | 直近1秒のキープレス数 |
| `Idle` | 最後の入力からの経過時間 |
| `Status` | 状態表示 |

## ステータス表示

- `[OK]` - 通常動作
- `[IDLE]` - 30秒以上入力なし
- `[ZONE]` - 高速（80km/h以上）
- `[OVER]` - 超高速（100km/h以上）

## パラメータ調整

`dev_speed_meter.py` の先頭で調整可能：
```python
SMOOTH_ALPHA = 0.3        # 平滑化係数（0.1-0.5推奨）
SPEED_SCALE = 12          # スピード感度（変更で上昇速度調整）
ALERT_THRESHOLD = 100     # アラート閾値
```

- `SMOOTH_ALPHA` が小さいほど変動が緩やか
- `SPEED_SCALE` が大きいほどスピード上昇が早い

## 動作

- キーボード入力を監視（全キー対象）
- 1秒ごとにキープレス数をカウント
- EMA平滑化で急激な変動を抑制
- 100ms更新でリアルタイム表示

## 要件

- Python 3.7+
- Windows / macOS / Linux
- pynput ライブラリ
